# برنامج أجور الراجحي - نسخة تسليم Desktop

برنامج Desktop مخصص لإدارة عمال الشركة وتوليد ملف أجور TXT لبنك الراجحي.

## ما الذي يفعله البرنامج؟

- إدارة العمال والموظفين.
- حفظ بيانات العمال داخل البرنامج بدل رفع Excel كل مرة.
- استيراد أولي من ملف Excel الخاص بالعميل.
- تعديل الراتب والبدلات لكل عامل بشكل منفصل.
- تعديل جماعي عند الحاجة.
- حفظ مسير كل شهر والرجوع له لاحقًا.
- توليد ملف TXT لبنك الراجحي.
- تشغيل على 3 أجهزة داخل نفس الشبكة عبر قاعدة بيانات مشتركة.

## التشغيل المحلي للتجربة

```bash
pip install -r requirements.txt
python run.py
```

## استخراج EXE بدون Visual Studio

1. ارفع محتويات المشروع على GitHub.
2. افتح تبويب Actions.
3. شغل Build Windows EXE.
4. حمل Artifact باسم `RajhiWages-Windows-EXE`.
5. فك الضغط وستجد `RajhiWages.exe`.

## التشغيل على أكثر من جهاز

### نفس البيانات على 3 أجهزة داخل نفس المكتب

1. اختر جهاز رئيسي.
2. أنشئ فولدر مشترك على الجهاز الرئيسي مثل:
   `D:\RajhiWagesData`
3. أعطِ صلاحية قراءة وكتابة لباقي الأجهزة.
4. من البرنامج افتح تبويب الإعدادات.
5. اختر مسار قاعدة البيانات المشتركة داخل الفولدر المشترك.
6. كرر نفس المسار في باقي الأجهزة.

## أول تشغيل عند العميل

1. افتح البرنامج.
2. من إعدادات المنشأة أدخل:
   - اسم المنشأة.
   - بنك المنشأة RJHI.
   - رقم المنشأة في البنك.
   - آيبان حساب الشركة.
   - رقم منشأة وزارة العمل.
3. اذهب إلى تبويب العمال.
4. اضغط استيراد Excel العميل.
5. بعد الاستيراد راجع عدد العمال والإجمالي.
6. اذهب إلى توليد TXT.
7. أدخل تاريخ المسير وتاريخ الخصم.
8. اضغط توليد TXT.

## ملاحظات مهمة

- البرنامج مبني على بنية ملف Wage Details المرسل من العميل.
- ملف TXT الناتج يعتمد على نفس ترتيب الحقول الظاهر في ملف العميل.
- في حال نظام البنك طلب توقيع رقمي `.sign.txt` أو اعتماد من بوابة البنك، فهذا يتم من البنك نفسه وليس من البرنامج.
- البرنامج ينتج ملف TXT، وأي توقيع رقمي رسمي يتم من منصة البنك إن كان مطلوبًا.
